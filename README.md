# predict-interface

## environment

```
node >= 18.18.2
npm >= 9.8.1
```

## install

```bash
yarn install
```

## development

```bash
# default is evm-testnet
yarn dev

# use mode to specify the env
yarn dev --mode production
```

## build

```bash
# for evm-testnet
yarn build:testnet

# for evm
yarn build
```
